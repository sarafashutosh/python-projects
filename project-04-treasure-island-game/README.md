# 🏝️ Treasure Island - Python Project

## 📌 Overview

**Treasure Island** is a beginner-friendly text-based adventure game 
built in Python.

The goal of the game is simple: **find the treasure** 
by making the correct choices throughout the journey. 
Each decision leads to different outcomes, 
including winning the game or facing various "game over" scenarios.
---
## 🎯 Learning Objectives

This project focuses on core Python fundamentals, including:

* Conditional statements (`if`, `elif`, `else`)
* Logical operators (`and`, `or`, `not`)
* User input handling
* Control flow
* String manipulation
* Nested conditions

---

## 🕹️ How the Game Works

The player is prompted to make decisions at different stages of the adventure:

1. Choose a direction (left or right)
2. Decide whether to wait or swim
3. Select a door (red, blue, or yellow)

Each choice determines the outcome of the game.

---
## 🧠 Example Gameplay

```
You're at a crossroad. Where do you want to go? Type "left" or "right"
> left
You've come to a lake. There is an island in the middle.
Type "wait" to wait for a boat. Type "swim" to swim across.
> wait
You arrive at the island unharmed. There is a house with 3 doors.
One red, one yellow, and one blue. Which colour do you choose?
> yellow
You found the treasure! You Win!
```
